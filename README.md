1. Project Title
Blossom Scoop Shop

- Project Description
This project is a web-based Ice Cream Shop Management System developed for Blossom Scoop Shop. It allows customers to browse sorbet, gelato, and mochi ice cream menus, can see chart and can login, register and logout.
